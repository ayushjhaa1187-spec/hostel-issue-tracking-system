# hostel-issue-tracking-system
**Centralized digital portal for streamlining student grievances and facility maintenance in hostels.**

## Story
"The fan was rattling for weeks. His paper complaint sat in a dusty binder. He opened the Portal. One submission. One photo. The warden saw it. Two hours later, the repairman arrived. Quiet at last. The hostel finally worked. Secure."

## Tech Stack
- **Frontend:** React, Tailwind CSS
- - **Backend:** Node.js, Express
  - - **Database:** MongoDB
    - - **Authentication:** JWT, Bcrypt
